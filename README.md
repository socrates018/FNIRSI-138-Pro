# FNIRSI-138 Pro Firmware (Fork)

This repository is a fork of DSO-138 firmware, adapted for the FNIRSI-138 Pro.

## What was changed

- Adapted to PlatformIO.
- Arduino `.ino` project layout kept compatible with PlatformIO build flow.
- Folder structure cleaned up/fixed.

PlatformIO note about `.ino` to `.cpp` handling:
https://docs.platformio.org/en/latest/faq/ino-to-cpp.html
